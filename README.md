# Scenerio for HackBright Capstone
Repo for Hackbright capstone class.  The scenerio I am addressing is deploying an internally hosted password manager and then automating the configuration of company owned resources to only use this service.  Much of the drive for this is to reduce the chance of enterprise credentials being leaked by users saving them to their own personal accounts.  Ie Google Chrome's Password Manager, Apple's iCloud Keychain, Windows Credential Manager, etc, etc....

### Resources
BitWarden - I selected this as the password manager due to ability to selfhost while still having a robust ecosystem of documentation and industry support.  

Azure - Chosen over AWS, GCP, selfhosting, etc, etc...  Mainly due to wanting to learn more about this specific cloud provider and also because I didn't want to risk losing this lab environment if my local environment died.
